第二种方式使用脚手架，这个脚手架中正常使用es6语法

1.git init 初始化一个空的本地git仓库
配置
git config --global user.name 'Stacey-TL'
git config --global user.email '2502421591@qq.com'

2.git add<file>添加文件，添加文件后会形成一个队列
【如果要指定某一类型的所有文件，git add *.html】
【如果是所有文件就是git add .或者git add *】
3.git status查看是否有添加修改的状态
4.git commit提交到本地仓库中
5.git push把本地仓库的数据pull(推送)到远程仓库中
